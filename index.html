Privacy Policy

Last updated: April 2026

Angus Health Tracker ("the App") is a personal health tracking application built for private, single-user use. This policy explains what data is collected and how it is used.

Data Collected

The App collects the following data, all of which is entered by or generated by the user:

  • Body measurements (weight, waist circumference)
  • Nutrition logs (meal descriptions, protein and fiber estimates)
  • Training session records
  • Sleep and recovery data (via Whoop API integration)
  • Supplement and habit logs
  • Progress benchmarks and photos (optional)

How Data Is Used

All data is used exclusively to display personalised health insights and programme tracking to the authenticated user. Data is never sold, shared with third parties, or used for advertising purposes.

Third-Party Integrations

The App connects to the Whoop API to retrieve recovery, sleep, and strain data with the user's explicit authorisation. This connection is governed by Whoop's own privacy policy. The App uses Anthropic's Claude API to generate on-demand health insights from the user's own logged data. No personal data is retained by Anthropic beyond the scope of individual API requests per Anthropic's API data policy.

Data Storage

User data is stored in a Supabase PostgreSQL database with row-level security enabled. Only the authenticated account holder can access their own data.

Data Retention and Deletion

Users may delete their account and all associated data at any time through the App settings.

Contact

This is a private personal application. For any questions, contact the app owner directly.
